# Binance Service

## This service allows to retrieve current mid Bitcoin price with applied service commission

`GET /price`

## Development

`node` version `18.16.1`

```
nvm install 18.16.1
```

```
nvm use 18.16.1
```

The first time, you will need to run

```
npm install
```

Then start the server with

```
npm run start
```

Tests could be run via

```
npm run test
```
